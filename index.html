Integration Guide

Connect your local ML model by configuring the endpoint above. The frontend sends requests in this format:

POST {endpoint}
Content-Type: application/json

{
  "model": "{model_name}",
  "prompt": "{user_message}",
  "context": {
    "mood": "{current_mood}",
    "history": [...previous_messages],
    "journal": "{recent_journal_entry}"
  },
  "temperature": {temperature},
  "stream": false
}

Expected response: { "response": "..." } or OpenAI-compatible choices[0].message.content
