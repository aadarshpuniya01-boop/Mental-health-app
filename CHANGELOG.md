## [1.0.0] — 2025

### Added
- Mood check-in panel with emoji/label selection
- AI chat companion with configurable LLM backend (OpenAI-compatible & Ollama)
- Guided journal with optional AI mood analysis
- Interactive 4-7-8 breathing exercise with animated ring
- Insights dashboard with mood trend visualization
- Model configurator modal (endpoint, model name, API type, system prompt, temperature)
- Fallback responses when no model is connected
- Responsive sidebar navigation (Dashboard, Journal, Exercises, Resources, Settings)
- Full-screen chat mode
